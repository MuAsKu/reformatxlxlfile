# reformatxlxlfile

Конвертер Excel-шахматки квартир в CSV с определением статуса по цвету ячейки.

## Установка

```bash
npm install
```

## Запуск

```bash
node excel-to-csv.js "файл.xlsx" [выход.csv]
```

## Формат CSV

Экспорт содержит 11 колонок: номер помещения, цена за м², валюта, этаж, подъезд, площадь, статус, кол-во комнат, название ЖК, название дома, описание помещения.

## Статусы по цвету

- Белый / без заливки → Свободно
- Синий → Забронировано
- Фиолетовый → Недоступно
